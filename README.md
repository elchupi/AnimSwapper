# AnimSwapper

FFXIV Dalamud plugin for race/job animation swaps and Glamourer territory automation.

## Features

- **Race Animation Swaps** — swap walk, run, and idle animations between races via Penumbra IPC
- **Female Animation Toggle** — per-rule option to use female variants of animations
- **Glamourer Territory Automation** — auto-apply Glamourer designs when entering specific zones, with revert on leave
- **Duty Glamour** — set a global glamour design for all duties, with per-territory overrides

## Requirements

- [Dalamud](https://github.com/goatcorp/Dalamud) (API level 15+)
- [Penumbra](https://github.com/xivdev/Penumbra) — for animation swaps
- [Glamourer](https://github.com/Ottermandias/Glamourer) — for glamour automation (optional)

## Installation

Add as a dev plugin or install from a custom Dalamud repo.
